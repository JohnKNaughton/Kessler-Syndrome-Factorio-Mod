KESSLER SYNDROME: Scrap Above Fulgora — Factorio Space Age Mod
==============================================================

CONCEPT
-------
Centuries of industrial activity have left Fulgora's orbit choked with
debris. The routes to the planet are scattered with small glowing chunks
of junk, growing denser as you approach. In orbit itself, the belt is
overwhelmed — a true Kessler Syndrome event frozen in time.


WHAT THIS MOD ADDS
------------------
ONE ASTEROID ENTITY:
  • Scrap Asteroid — small, light-green glow, same toughness as base
    game asteroids of the same size

ONE CHUNK ITEM:
  • Scrap Asteroid Chunk — green-tinted icon, distinct from metallic chunks

CRUSHING OUTPUT (crusher reads this directly — no switchable recipes):
  • Scrap (4–9)                         — always
  • Depleted Uranium Fuel Cell (1–2)    — 55% chance
  • Uranium-235 (1)                     — 2% chance

SPAWNING:
  • Transit routes (Nauvis, Vulcanus, Gleba → Fulgora):
    Sparse near the origin, building steadily, peaking in a dense
    cluster right as you approach Fulgora. Easy to deal with early,
    increasingly demanding as you near the planet.

  • Fulgora orbit:
    5× the combined probability of all other asteroid types in orbit.
    Overwhelmingly scrap. Build extra crushers before you arrive.


HOW TO INSTALL
--------------
1. Press Win+R → type %appdata%\Factorio\mods → Enter
2. Copy the kessler-syndrome_1.0.0 folder in there
3. Launch Factorio → Mods → enable Kessler Syndrome
4. Requires Space Age DLC


TIPS
----
- The green glow makes scrap asteroids easy to spot at a distance.
- Every crush has a 55% chance of a depleted fuel cell and a 2% chance
  of uranium-235. With orbital volume, both add up over time.
- Scrap pairs naturally with Fulgora's recycler production chains.


COMPATIBILITY
-------------
Orbit patch scales dynamically to whatever asteroid probabilities
Fulgora already has, so other mods adding Fulgora asteroids are fine.
Transit connections are patched by explicit name with a full fallback scan.
